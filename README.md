# 南昌大学 AI 阅卷助手 (NCU AI Grading Assistant)

这是一个专为南昌大学在线阅卷系统 (`yuejuan.ncu.edu.cn`) 打造的 Chrome/Edge 浏览器 AI 辅助阅卷扩展。它能够帮助教师通过大语言模型 Vision 能力自动识别并阅卷、智能绘制红色手绘评语与 ✓/✗/⍻ 状态印章，并且自动将小题得分回填并提交，大幅提高阅卷效率。

## 核心特性
* 🤖 **AI 自动评分与点评**：自动将学生答卷画布转化为图片，通过 LLM API（如 Qwen 系列视觉大模型、Google Gemini 等）智能识别作答内容（包括手写文字、决策树、决策表等）并进行分析、打分和撰写精炼点评。
* 📸 **图片/截图标准答案支持**：除文本评分标准外，支持各小题上传、拖拽、或直接从剪贴板粘贴（Ctrl+V）标准答案截图。大模型可在单次请求中自动比对学生答卷与标准答案图片，实现极致精准的判分。
* 🎨 **手绘风格红色涂鸦**：自动切换画笔工具，直接在学生答卷画布（`.canvas-sketcher`）上书写纯红色的点评文本和实得分数。
* 🎯 **位置定位与自动盖章**：根据 AI 检测的视觉中心，自动盖下 ✓、✗ 或 ⍻ 印章。
* 📝 **确认与微调缓冲区**：AI 评阅后会首先弹出一个 Glassmorphism 磨砂玻璃风格的对话框。教师可对每个小题的实得分数、对错状态、批语内容进行任意微调修改，确认后再一键绘制到画布并保存。
* 📊 **分数自动录入**：自动精确计算所有小题的总分并快速回填至系统打分输入框，智能适配 EasyUI 等各种输入控件并支持自动提交分数。
* 🧭 **左右排布智能折行**：当答题纸上的小题成左右分栏排列时，批注文字和得分会自动在各自的水平列区域内折行并保留安全间距，杜绝多题评语重叠覆盖。

## 安装指南
1. 下载本项目代码（点击本页面右上角的 `Code` -> `Download ZIP` 解压，或使用 Git 克隆本仓库到本地）。
2. 打开 Chrome 或 Edge 浏览器，进入扩展程序页面（在地址栏输入 `chrome://extensions/` 访问）。
3. 开启页面右上角的 **“开发者模式”**（Developer mode）开关。
4. 点击左上角的 **“加载已解压的扩展程序”**（Load unpacked），选择解压后的整个项目文件夹（包含 `manifest.json` 的目录）。
5. 安装成功后，点击浏览器右上角的拼图图标（扩展程序），找到“南昌大学 AI 阅卷助手”，即可看到配置弹窗。

## 配置与连接
点击插件图标，在弹窗中配置大模型连接参数：
1. **API Provider**：选择 API 提供商，支持 `OpenAI / vLLM (Compatible)`、`SiliconFlow (硅基流动)`（内置了最新的视觉大模型默认参数）或 `Google Gemini`。
2. **API URL**：填写大模型的 API 端点地址。
3. **API Key**：填写您的大模型 API 密钥。
4. **Model Name**：填写调用的 Vision 视觉大模型名称（支持视觉的多模态模型才能处理答卷图片）。
5. **当前阅卷课程**：正在评阅的课程名称，用于优化 AI 评分口吻和角色设定（通常扩展会自动从页面检测并自动填入，无需手动填写，仅在需要手动重写时修改）。
6. 点击 **“保存配置”**。

## 使用方法
1. 登录南昌大学在线阅卷系统，进入阅卷评阅页面。
2. 页面右下角会自动加载一个磨砂玻璃风格的“AI 阅卷助手”悬浮面板（可拖拽和收起）。
3. **小题配置**：
   * 在配置栏中，输入您需要评阅的各小题分值（最高分数）以及“评分标准 (文本)”。评分标准越详细，AI 批改的准确度就越高。
     > 💡 **公式题输入技巧 (推荐 LaTeX)**: 
     > 如果评分标准中涉及复杂的数学、物理公式，**强烈建议使用标准的 LaTeX 语法输入文本评分标准**。大模型在解析 LaTeX 公式时匹配精度最高，并能支持乘法交换律等数学等价性判断。
     > * *推荐示例 (LaTeX)*: `\frac{3}{4}mR^2\omega^2`  
     > * *避免输入 (普通文本)*: `3/4 * m * R^2 * w^2` （容易因星号、斜杠、字母 w 的手写体歧义导致 AI 扣分）
     > * *根号与积分*: 推荐 `\sqrt{x^2 + y^2}` 避免输入 `根号下x方加y方`。
   * **评分标准 (图片/截图)**：支持为每个小题配置图形化标准答案。您可以：
     * **点击/拖拽上传**：点击卡片虚线区弹出文件浏览器，或者直接将图片文件拖入卡片中，卡片会感应并自动解析。
     * **剪贴板直接粘贴**：点击或使用 Tab 键使卡片区域获得焦点（此时卡片会有蓝色高亮框反馈），直接按 `Ctrl+V` (Windows) 或 `Cmd+V` (Mac) 即可直接粘贴屏幕截图。
     * **预览与删除**：上传成功的图片会呈 16:9 精美圆角预览，鼠标悬停时会展现半透明遮罩与“删除图片”按钮，可轻松清理与更换。
   * 可以在大题题干栏输入这道大题的整体业务规则或背景描述（例如决策表相关的折扣率规则）。
   * **配置与图片自动恢复**：只要不更换浏览器，所有小题配置（包含文本标准答案和**已上传的标准答案图片**）在下次访问时都会自动恢复（系统会自动按 `examId` 与大题题目自动保存与加载，由于已在扩展中申请 `unlimitedStorage` 权限，图片缓存不受 5MB 容量限制，无需重复上传）。
4. **开始阅卷**：
   * 配置完成后，点击 **“🤖 智能阅卷 (AI)”** 按钮。
   * AI 识别完成后，将弹出批阅结果确认框。检查分数与评语，确认无误后点击 **“确认并写入”**，印章与评语将被手绘写入画布中，总分将自动填入右侧打分框。
5. **测试功能**：
   * 点击 **“🎨 测试画布批注”** 按钮可以测试当前浏览器是否能成功切换画笔并模拟手绘（如成功，将在画布中心画出 ✓、✗ 评语和 5 分）。
